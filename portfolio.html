Press Rekord

  • May 2021
  • Windows PC
  • My Role: Main Programmer
  • Project Page
  • Project Repository

Press Rekord is a 2D action-adventure game set in a curious underground world filled with monsters both friendly and nefarious. The game follows the story of Kat, an up-and-coming urban explorer live streamer, who finds herself in a mysterious subterranean town in an abandoned subway station, filled with bright neon lights and even more colourful characters. While only being a short demo of a potential game, Press Rekord has a lot of content to mess around with, and features cool tools and systems for the player to interact with. Players can follow 'quests' and use a map tool to find their way through the labyrinthine tunnels outside of the main town. Kat can also perform different types of attacks.

The short demo was created by a team of students for their final year project for the Bachelor of Arts in Game Design Program at TUDublin. Team Members included Megan Grant, Michael Ward, Aaron Kennedy, Tabeeb Choudhury, and me as the main programmer. It was a difficult and unprecedented time for students in recent years, as it was the year in which COVID 19 forced the world into isolation. We adapted to the new challenge and were able to complete the project.

The features I am most proud of was my implementation of the quest systems, the foundations of the combat system, the map and user interfaces including the dialogue system, and my favourite contribution, the final boss encounter.

The team and I learned a lot through our struggles, and we gained experience and skill both in our technical abilities, but also our interpersonal communication and finding internal motivation to keep going even when the world was struggling under the weight of the pandemic.

Ubuntu Bash Backup and Auditing System

  • May 2022
  • Ubuntu Linux
  • Github Repository

As part of my Architecture, Operating Systems, and Networking module assessment, I created a backup and auditing system using Bash scripts in an Ubuntu Virtual Machine.

This was a challenging assignment, as I had no previous experience with using the command line for anything more complicated than basic troubleshooting for my personal computer. I used crontab for scheduling tasks using auditd and performing system health checks. Additionally, I decided to attempt to add an automated email service, though this went beyond the assignment brief.

It was a great learning experience, that helped me become comfortable with using the command line in a Linux environment, and has made me consider swapping over to a Linux Operating System for my next machine!

Photo

  • April 2020
  • Windows PC
  • Programming, Design
  • Project Page

During the second semester of my third year, I was taking a module in rapid prototyping, where assessment was based on our ability to create working prototypes within short 2 week sprints.

At the start of each sprint, we were given a high level brief regarding the genre of game we were to create, these included strategy, racing, and an individual assignment.

For the individual Assignment, I created Photo, a 3D puzzle platformer proof-of-concept prototype. In the game, the player has the ability to use a magic camera to swap ‘properties’ of objects within the game world to solve puzzles.

This was an enjoyable assignment, as it gave me total creative freedom to experiment, and the chance to learn new technologies and skills within the Unity game engine.

Python Command Line Library

  • May 2022
  • Student Project
  • Github Repository

The Python Command Line Library project was a small final project for the Object-Oriented Programming module for the Fundamentals of Computing pre-Masters course at TUDublin. It was a small-scale assignment and were given a couple of options as to what we wished to create. My initial intention was to create a visual application using buttons using TKinter and other modules, however, due to time constraints with other assignments, I was unable to justify the extra time towards it.

The project consists of a command line interface that allows users to rent books and return them, as well as signing members up to the virtual 'library'. It is a simple program, but it serves to demonstrate my understanding of the fundamentals of programming in Python.

Although I had previous experience with other languages such as C# for Unity, and C++ as I was introducing myself to graphics programming, it was fun to be introduced to another language and I found it quite easy to pick up. I feel more inspired to try my hand at other languages now and to find cool projects to work on.

Gardiens

  • December 2019
  • Student Project
  • Github Repository

Gardiens was a short Unity Mobile game which aimed to act as an Electronic Adherence Monitor (EAM), a type of software or physical device which aims to assist patients to administer their medication according to their doctors instructions.

The project used gamification to attempt to build compliance in the patient, through association with the management of a virtual garden / terrarium. Users could plant seeds, water flowers, and combine different seeds to create new flowers. For their various medications, users could set up multiple reminders per day, which would send discreet notifications to their device.

I acted as a programmer for this project, and contributed to the reminder system within the game.
